# AC/DC: AWS Cloud/DevOps Culture

## Install

Clone the project

    $ git clone git@github.com:chaordic/act-acdc.git

Install dependencies

    $ cd act-acdc
    $ npm install

## Getting Started

Running the project

    $ node index.js

You should see the following message on your console

    AC/DC is up and running!

Point you browser to

    http://localhost:8080
